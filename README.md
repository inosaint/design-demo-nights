# Design Demo Nights

A no-build static site for documenting Design Demo Nights events.

## Edit content

- Event and demo content lives in `script.js`.
- Future events render as upcoming cards with an apply CTA.
- Past events render as archive cards with demo detail dialogs.
- Local images live under `assets/events/<event-slug>/`.

## GitHub Pages

Serve from the repository root. No build command is needed.
